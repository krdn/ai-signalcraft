# Feature Landscape

**Domain:** 한국 공인 여론 분석 자동화 파이프라인 (Korean Public Figure Opinion Analysis)
**Researched:** 2026-03-24
**Overall confidence:** MEDIUM-HIGH

## Table Stakes

사용자가 당연히 기대하는 기능. 없으면 제품으로서 성립 불가.

| Feature | Why Expected | Complexity | Notes |
|---------|--------------|------------|-------|
| **멀티 플랫폼 데이터 수집** | 빅카인즈(뉴스), 썸트렌드(소셜+커뮤니티), VAIV(종합) 모두 다채널 수집 지원. 단일 소스만으로는 여론 파악 불가 | High | 네이버 뉴스, 유튜브, X, DC갤러리, 에펨코리아, 클리앙 — 각각 수집 전략 상이 (API vs 스크래핑) |
| **키워드/인물 기반 검색** | 모든 여론 분석 도구의 기본 진입점. 빅카인즈·썸트렌드 모두 키워드 검색이 핵심 UX | Low | 검색어 + 기간 + 소스 필터 조합 |
| **감성 분석 (Sentiment Analysis)** | 썸트렌드의 긍부정 분석, Brandwatch의 AI 감성 분석 등 모든 경쟁 제품이 제공. 여론 분석의 핵심 지표 | Medium | 한국어 감성 분석은 KoBERT 등 딥러닝 기반이 lexicon 기반보다 정확. 교착어 특성상 형태소 분석 필수 |
| **시계열 트렌드 차트** | 빅카인즈의 키워드 트렌드, 썸트렌드의 추이 분석 등 시간 축 시각화는 기본. 여론의 변화 방향 파악에 필수 | Low | 일별/주별/월별 언급량 + 감성 비율 추이 |
| **연관어/키워드 분석** | 빅카인즈의 연관어 분석, 썸트렌드의 연관어 변화 분석. 어떤 맥락에서 논의되는지 파악하는 기본 기능 | Medium | 워드클라우드 + 네트워크 그래프 형태 시각화 |
| **데이터 내보내기 (Export)** | 빅카인즈는 엑셀/PDF/Word 내보내기 지원. 분석 결과를 보고서로 활용하려면 필수 | Low | CSV, PDF 리포트 형식 |
| **기간/소스 필터링** | 모든 경쟁 도구의 기본 필터. 날짜 범위, 플랫폼별 필터 없으면 데이터 탐색 불가 | Low | 날짜 범위 선택기 + 플랫폼 체크박스 |
| **수집 데이터 저장 및 이력 관리** | 분석 재현성과 비교 분석을 위해 수집 데이터의 영속적 저장 필수 | Medium | PostgreSQL/MongoDB에 수집 메타데이터 + 원본 저장 |
| **대시보드 기본 시각화** | Brandwatch의 커스터마이즈 대시보드, Meltwater의 실시간 대시보드 등 시각적 결과 제공은 기본 | Medium | 차트(라인, 바, 파이) + 요약 카드 + 테이블 |
| **팀 멀티유저 접근** | 3~10명 분석팀이 동시 접근하는 요건. 경쟁 도구들도 팀 기능 기본 제공 | Medium | 인증 + 권한 관리 + 동시 접속 |

## Differentiators

경쟁 우위를 만드는 기능. 기대하지 않지만 있으면 큰 가치.

| Feature | Value Proposition | Complexity | Notes |
|---------|-------------------|------------|-------|
| **AI 종합 분석 리포트 자동 생성** | 빅카인즈·썸트렌드는 수치/그래프만 제공하고 해석은 사용자 몫. AI가 수집 데이터를 종합하여 "전략적 인사이트가 포함된 텍스트 리포트"를 자동 생성하면 분석팀 시간 대폭 절감 | High | Claude/GPT로 8개 분석 모듈(여론 구조, 집단별 반응, 감정/프레임, 메시지 효과, 리스크, 기회, 전략, 지지율) 결과를 종합 리포트화. 이것이 AI SignalCraft의 핵심 차별점 |
| **프레임 분석 (Frame Analysis)** | 단순 긍/부정을 넘어 "어떤 프레임으로 논의되는가" 분석. 정치 전략에서 프레임 전쟁 파악은 핵심이나 기존 도구는 미제공 | High | "도덕성 프레임" vs "능력 프레임" vs "안보 프레임" 등 분류. LLM 기반 분류가 적합 |
| **메시지 효과 분석** | 어떤 메시지/발언이 여론에 긍정/부정 영향을 미쳤는지 식별. 캠페인 전략 수립에 직접 활용 가능한 인사이트 | High | 특정 이벤트/발언 전후 감성 변화 분석 + AI 해석 |
| **리스크/기회 분석** | 현재 여론에서 Top 리스크와 확장 가능한 긍정 요소를 자동 식별. 전략 의사결정 속도 향상 | High | LLM이 수집 데이터에서 리스크 요인 추출 + 영향도 평가 |
| **위기 대응 시나리오 생성** | 리스크 발생 시 대응 시나리오를 AI가 자동 생성. 기존 도구에는 없는 기능으로 전략팀에 즉시 활용 가능 | Medium | LLM 프롬프트 기반. 리스크 분석 결과를 입력으로 시나리오 3개 생성 |
| **집단별 반응 세분화** | 플랫폼별(네이버 vs 유튜브 vs 커뮤니티), 정치 성향별 반응 차이 분석. 기존 도구는 전체 감성만 제공하는 경우 다수 | High | 플랫폼별 분리는 수집 시 태깅으로 가능. 성향/연령 추정은 LLM 기반 추론 필요 |
| **AI 지지율 추정** | 전통 여론조사 없이 온라인 데이터 기반 지지율 추정. VAIV가 유사 시도했으나 정확도 이슈 있음. 참고 지표로서 가치 | Very High | 온라인 여론 ≠ 실제 지지율. "AI 추정치"로 명확히 표기하고 신뢰구간 제시 필수. 보정 모델 필요 |
| **경쟁 인물 비교 분석** | 두 인물의 여론을 나란히 비교. 썸트렌드의 키워드 비교와 유사하나 인물 단위 종합 비교는 차별화 | Medium | 동일 기간 수집 후 병렬 대시보드. Share-of-Voice 개념 적용 |
| **전략 도출 및 권고** | 분석 결과를 바탕으로 "타겟/메시지/콘텐츠/리스크 대응" 전략을 AI가 제안. 분석 → 실행 간 격차를 줄이는 핵심 가치 | High | LLM 프롬프트 체인. 모든 분석 모듈 결과를 종합하여 전략 도출 |
| **다중 AI 모델 유연 전환** | 분석 항목별 최적 모델 선택(Claude는 긴 맥락 분석, GPT는 구조화된 출력 등). 비용/품질 균형 최적화 | Medium | AI SDK 등으로 모델 추상화. 분석 모듈별 모델 매핑 설정 |
| **분석 히스토리 비교** | 동일 인물에 대한 과거 분석과 현재 분석을 비교하여 여론 변화 추적. 시간에 걸친 전략 효과 측정 | Medium | 분석 결과 DB 저장 + 시점 간 diff 시각화 |

## Anti-Features

의도적으로 만들지 않을 기능.

| Anti-Feature | Why Avoid | What to Do Instead |
|--------------|-----------|-------------------|
| **실시간 자동 수집 (스트리밍)** | 비용 폭증 (AI API + 인프라), 수동 트리거로 결정 완료. 3~10명 팀 규모에서 실시간은 과도 | 수동 트리거 방식 유지. 필요 시점에 집중 수집하여 비용 관리 |
| **모바일 네이티브 앱** | 개발 복잡도 2배. 대시보드 확인은 웹으로 충분 | 반응형 웹 대시보드로 모바일 접근 지원 |
| **자연어 대화형 인터페이스 (챗봇)** | 대시보드 기반 UX로 결정 완료. 챗봇은 UX 복잡도 증가 대비 가치 불명확 | 구조화된 대시보드 + AI 생성 리포트 텍스트로 충분 |
| **해외 여론 분석** | 한국 여론에 집중하는 것이 프로젝트 범위. 해외 소스 추가 시 수집/분석 복잡도 급증 | 한국 플랫폼(네이버, 유튜브 한국어, X 한국어, 국내 커뮤니티)에 집중 |
| **여론조사 대체 주장** | AI 지지율 추정은 참고 지표이지 과학적 여론조사 대체 불가. 과도한 신뢰성 주장은 법적/윤리적 리스크 | "AI 추정 참고치"로 명확히 레이블링. 신뢰구간 표시. 공식 여론조사와 병기 권장 |
| **개인정보 수집/프로파일링** | 댓글 작성자 개인 식별·추적은 개인정보보호법 위반 가능. 윤리적으로도 부적절 | 익명화된 집계 데이터만 분석. 개인 단위 추적 기능 배제 |
| **자동 댓글/게시글 작성** | 여론 조작 도구로 변질 위험. 분석 도구의 신뢰성 훼손 | 분석 및 인사이트 제공에만 집중. "읽기 전용" 원칙 |
| **범용 마케팅/브랜드 분석** | Brandwatch, Meltwater 등 글로벌 도구와 직접 경쟁하게 됨. 정치/공인 분석이라는 니치에 집중해야 차별화 | 공인(정치인, 연예인, 기업인) 여론 분석에 특화. 범용 브랜드 모니터링은 범위 밖 |

## Feature Dependencies

```
[데이터 수집 파이프라인]
  ├── 네이버 뉴스 수집기
  ├── 유튜브 댓글 수집기
  ├── X(트위터) 수집기
  └── 커뮤니티 수집기 (DC, 에펨코리아, 클리앙)
       │
       ▼
[데이터 저장 (DB)]
       │
       ▼
[기본 분석 엔진]
  ├── 감성 분석 ──────────────────────┐
  ├── 연관어/키워드 분석               │
  └── 시계열 트렌드 분석               │
       │                              │
       ▼                              ▼
[AI 심층 분석 모듈] ◄── 감성 분석 결과 필요
  ├── 프레임 분석 ◄── 연관어 분석 결과 활용
  ├── 메시지 효과 분석 ◄── 시계열 + 감성 결합
  ├── 집단별 반응 분석 ◄── 플랫폼별 감성 세분화
  ├── 리스크 분석
  ├── 기회 분석
  └── AI 지지율 추정 ◄── 모든 분석 결과 종합 필요
       │
       ▼
[전략 도출 모듈]
  ├── 전략 권고 ◄── 리스크 + 기회 + 프레임 분석 결합
  ├── 위기 대응 시나리오 ◄── 리스크 분석 결과 필요
  └── 승리 확률 시뮬레이션 ◄── AI 지지율 + 전략 분석 결합
       │
       ▼
[종합 리포트 생성] ◄── 모든 분석 모듈 결과 필요
       │
       ▼
[웹 대시보드]
  ├── 기본 시각화 (차트, 테이블)
  ├── 리포트 뷰어
  ├── 분석 히스토리
  ├── 경쟁 인물 비교
  └── 데이터 내보내기
       │
       ▼
[팀 기능]
  ├── 인증/권한 관리
  └── 멀티유저 동시 접근
```

### 핵심 의존 관계 요약

| 선행 기능 | 후행 기능 | 관계 |
|-----------|-----------|------|
| 데이터 수집 파이프라인 | 모든 분석 기능 | 수집 없이 분석 불가 |
| 감성 분석 | 프레임 분석, 메시지 효과 분석, 집단별 분석 | 감성 데이터가 심층 분석의 입력 |
| 시계열 트렌드 | 메시지 효과 분석 | 시간 축 변화 데이터 필요 |
| 리스크 분석 | 위기 대응 시나리오 | 리스크 식별 후 시나리오 생성 |
| 모든 분석 모듈 | 종합 리포트 생성 | 리포트는 전체 결과 종합 |
| 기본 대시보드 | 경쟁 비교, 히스토리 비교 | 기본 시각화 위에 비교 기능 추가 |

## MVP Recommendation

### 반드시 포함 (Phase 1-2):
1. **데이터 수집 파이프라인** — 최소 2개 소스(네이버 뉴스 + 유튜브)로 시작. 핵심 데이터 흐름 검증
2. **감성 분석** — 여론 분석의 가장 기본적인 지표. AI 모델 파이프라인 검증 포함
3. **기본 대시보드 시각화** — 시계열 차트 + 감성 비율 + 키워드 테이블
4. **AI 종합 리포트 생성** — 이것이 핵심 차별점. 빠르게 검증해야 할 핵심 가치

### 다음 단계 (Phase 3-4):
5. **나머지 데이터 소스 추가** — X, DC갤러리, 에펨코리아, 클리앙
6. **프레임 분석 + 메시지 효과 분석** — 정치 전략 분석의 핵심 심층 기능
7. **리스크/기회 분석 + 전략 도출** — AI 분석의 전략적 가치 완성
8. **팀 멀티유저 기능** — 인증/권한 관리

### 후순위 (Phase 5+):
- **AI 지지율 추정** — 복잡도 매우 높고 정확도 보장 어려움. 다른 기능 안정화 후 실험적 추가
- **위기 대응 시나리오 생성** — 리스크 분석 안정화 후 추가
- **경쟁 인물 비교** — 기본 분석 완성 후 비교 뷰 추가
- **분석 히스토리 비교** — 충분한 히스토리 데이터 축적 후 의미 있음

### 명시적 제외:
- **승리 확률 시뮬레이션**: 정확도 보장 극히 어렵고, 잘못된 예측은 신뢰도 하락 초래. 장기 연구 과제로 분류
