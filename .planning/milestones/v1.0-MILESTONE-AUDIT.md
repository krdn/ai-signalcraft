### 신규 발견 이슈

#### INT-03: collectionJob status 'running' 미전환 (Low)

- **영향**: 파이프라인 모니터에서 수집 단계가 `pending` → `completed` 직행 (running 상태 없음)
- **위치**: `worker-process.ts` — 작업 시작 시 `updateJobProgress(dbJobId, {}, 'running')` 호출 없음
- **영향 요구사항**: FOUND-03, DASH-02
- **심각도**: Low — 기능 정상 작동, UX 개선 사항

#### INT-04: adminProcedure teamId 미포함 역할 검증 (Low)

- **영향**: 다중 팀 소속 사용자가 다른 팀의 admin 역할로 mutation guard를 통과할 수 있음
- **위치**: `apps/web/src/server/trpc/init.ts` adminProcedure — `teamMembers.userId` 조건만 사용, `teamMembers.teamId` 미포함
- **완화 요소**: downstream 핸들러가 `ctx.teamId`로 데이터를 스코핑하므로 실제 데이터 접근은 제한됨
- **영향 요구사항**: TEAM-02
- **심각도**: Low — 현재 단일 팀 사용 시 무영향

#### REPT-03 참고: pdf-exporter.ts UI 미연결

- `packages/core/src/report/pdf-exporter.ts`가 구현되어 있으나 `apps/web`에서 미사용
- UI의 PDF 내보내기는 `window.print()` 사용
- 기능적으로 PDF 내보내기가 가능하므로 요구사항은 충족
