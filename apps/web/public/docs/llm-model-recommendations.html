1.4 한국어 성능 종합 순위

🟢 상위권 (운영 추천) Claude Sonnet/Opus 4.6 > GPT-4.1 > Gemini 2.5 Pro/Flash
🟡 중위권 (보조 가능, 단순 분류·요약) Qwen3.x > Amazon Nova Pro > GPT-4.1 Mini > Gemini Flash-Lite
🔴 하위권 (단순 분류만 권장) DeepSeek V3/V4 > Grok-3 > Llama 4 Maverick > Mistral
기준 순위

구조화 출력 신뢰성 Claude = GPT-4.1 = Gemini Flash/Pro > 나머지
비용 효율 (캐싱 포함) DeepSeek V4 > Gemini Flash-Lite > Haiku 4.5 > Flash
긴 컨텍스트(32K+) Claude 전체, Gemini 전체, GPT-4.1, Llama 4 Maverick

🧩 Part 2 — 분석 모듈별 추천 매트릭스

2.1 모듈 그룹 분류

그룹 모듈 핵심 요구 특성

E: 완전 무료 Gemini CLI 전체 ✅ 유효 낮음
F: Claude+Gemini CLI CLI 혼합 ✅ 유효 낮음
G: DeepSeek 혼합 Stage1=DeepSeek Chat, 추론=DeepSeek Reasoner, 핵심=Sonnet 4.6 ⚠️ 개선 필요 — DeepSeek V4 전환으로 캐시 히트 시 최대 57% 절감 높음

3.2 시나리오 G 즉시 업그레이드 (DeepSeek V4 전환)

현재 G 시나리오의 deepseek-chat(V3)을 deepseek-v4로 교체 시:

  • 캐시 히트: $0.07/MTok → $0.03/MTok (57% 절감)
  • 캐시 미스: $0.27/MTok → $0.30/MTok (소폭 증가)
// packages/core/src/analysis/model-config.ts — 시나리오 G
// 변경 전
'macro-view':        { provider: 'deepseek', model: 'deepseek-chat' },
'segmentation':      { provider: 'deepseek', model: 'deepseek-chat' },
'sentiment-framing': { provider: 'deepseek', model: 'deepseek-chat' },
'message-impact':    { provider: 'deepseek', model: 'deepseek-chat' },

// 변경 후
'macro-view':        { provider: 'deepseek', model: 'deepseek-v4' },
'segmentation':      { provider: 'deepseek', model: 'deepseek-v4' },
'sentiment-framing': { provider: 'deepseek', model: 'deepseek-v4' },
'message-impact':    { provider: 'deepseek', model: 'deepseek-v4' },
주의: DeepSeek API에서 실제 모델 ID(deepseek-v4 또는 다른 이름)를 확인 후 적용할 것.
