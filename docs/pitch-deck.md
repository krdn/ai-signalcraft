## Slide 2: 문제

### 여론 분석의 현실

- 수동 클리핑에 **주 10~20시간** 투입
- 기존 도구는 감정 분석(긍정/부정)에서 멈춤
- 클라이언트/경영진이 묻는 질문: **"그래서 어떻게 대응해?"**
- 리포트 작성에 추가 **2~3일** 소요
- 네이버 뉴스, 유튜브는 되지만 **DC갤러리, FM코리아는 사각지대**

## Slide 5: 14개 분석 모듈 + 지식 인프라

### Stage 1: 초기 분석 (병렬)

- 거시 여론 구조 — 시간별 흐름, 변곡점
- 집단별 반응 — 지지층/반대층/유동층
- 감정·프레임 분석 — 희망/분노/두려움 강도
- 메시지 파급력 — 핵심 메시지 확산력

### Stage 2: 심화 분석 (순차)

- 리스크 지도 — 위기 가능성 + 대응 우선순위
- 기회 분석 — 긍정 이슈 + 포섭 가능 집단
- **전략 도출** — 타겟·메시지·콘텐츠·채널 전략
- 최종 요약 — 한 줄 핵심 인사이트

### Stage 4: 고급 분석 (선택)

- **지지율 추정** — 여론 데이터 기반 추정치
- **프레임 전쟁** — 주도 vs 위협 프레임 분석
- **위기 시나리오** — 3개 시나리오 + 대응책
- **승리 시뮬레이션** — 확률 + 핵심 전략

### 지식 인프라 (자동)

- **지식 그래프** — 인물·조직·이슈·프레임 간 관계를 네트워크로 시각화
- **시맨틱 검색** — 의미 기반 문서 검색으로 관련 기사/댓글 즉시 탐색
- **통합 리포트** — 전체 분석 결과를 PDF 전략 리포트로 자동 생성
