# 경쟁사 정밀 분석: Sometrend vs Adriel vs AI SignalCraft

> 작성일: 2026-04-16 | 리서치 방법: 사이트 직접 크롤링 + 공개 리뷰 데이터 수집

---

## 1. 서비스 포지셔닝 한 줄 요약

| 서비스                     | 한 줄 정의                                            |
| -------------------------- | ----------------------------------------------------- |
| **Sometrend** (some.co.kr) | 한국 SNS 빅데이터 기반 소비자 여론·트렌드 조회 서비스 |
| **Adriel** (adriel.com)    | 멀티채널 광고 운영(AdOps) 통합 데이터 플랫폼          |

## 2. Sometrend 심층 분석

### 서비스 개요

(주) 바이브컴퍼니가 운영하는 한국 소셜 빅데이터 분석 플랫폼. "사람들의 생각이 궁금할 때"를 슬로건으로 마케터·트렌드 리서처를 주요 고객으로 삼는다.

### 데이터 규모

| 채널        | 문서 수         | 기준 시점   |
| ----------- | --------------- | ----------- |
| 블로그      | 약 21억 건      | 2011년~현재 |
| 인스타그램  | 약 26억 건      | 2014년~현재 |
| 커뮤니티    | 약 4억 건       | 2014년~현재 |
| 네이버 뉴스 | 약 1.5억 건     | 2008년~현재 |
| 유튜브      | 약 7,000만 건   | 2018년~현재 |
| 언론사 뉴스 | 약 1.1억 건     | 2014년~현재 |
| **합계**    | **약 504억 건** | —           |

- 커뮤니티 게시판 7,000개 이상 수집 (뽐뿌, 네이트판, 중고나라 등)
- **썸트렌드 MCP** 출시 — LLM 연동용 API 최신 제공

### 핵심 분석 기능

- 키워드 추이 분석 (실시간 관심도 변화)
- 연관어 변화 분석 (다음 트렌드 예측)
- 긍부정 속성 분석 (브랜드 강점·약점 분류)
- 키워드 비교 분석 (최대 3개)
- 커뮤니티 분석 (최신 트렌드 커뮤니티 여론)
- AI 리포트 자동 생성 (플랜별 5~300회/월)
- 실시간 이슈 알림
- 인플루언서 검색
- 트렌드 랭킹 (소셜/뉴스)
- 브랜드 인사이트 분석 (별도 문의)

### 요금 구조

| 플랜                   | 월 가격              | 주요 특징                                           |
| ---------------------- | -------------------- | --------------------------------------------------- |
| **Free**               | 무료                 | 최근 1개월, 블로그·뉴스만, AI 리포트 5회            |
| **베이직**             | 29,700원             | 최근 1년, 커뮤니티 미포함, 리포트 100회             |
| **프리미엄**           | 149,000원            | 커뮤니티·유튜브 포함, 키워드 3개 비교, 리포트 300회 |
| **비즈니스(클라우드)** | 200만원~/월 (1~10인) | 2008년부터 전체, 성별·연령·관심사 세분화, 맞춤 기능 |

- 연간 결제 시 최대 41% 할인
- 베이직 34% / 프리미엄 17% 할인

### 강점

- 한국 소셜 데이터 최대 규모 (504억 건)
- 저렴한 진입 비용 (무료 플랜 제공)
- 직관적인 셀프서비스 UI
- MCP로 LLM 연동 지원 시작

### 약점

- 키워드 조회 중심 → 심층 AI 해석 없음
- 분석 커스터마이징 제한 (플랜이 정한 채널·기능만)
- "이런 말이 있다"까지 — "어떻게 대응하라"는 없음
- 데이터가 플랫폼 외부로 나감 (보안 민감 조직 부적합)

## 3. Adriel 심층 분석

### 서비스 개요

한국계 스타트업이 글로벌 시장에서 성장한 광고 운영(AdOps) 플랫폼. 2025년 Series B $1,300만 유치, 2027년 IPO 목표. 주 타겟: 광고 대행사·퍼포먼스 마케팅 팀.

### 핵심 기능

**데이터 통합**

- 650+ 데이터 소스 노코드 커넥터 (Facebook, Google, TikTok, LinkedIn, 카카오 등 광고 플랫폼)
- 코드 없이 실시간 데이터 파이프라인 자동 유지
- 맞춤형 드래그앤드롭 대시보드

**Creative Intelligence Suite**

- 어떤 키워드·색상·이미지·이모지가 전환을 높이는지 크리에이티브 단위 분석
- 크로스 플랫폼 광고 소재 성과 실시간 비교

**AI 기능**

- **AdGen AI** — 광고 소재 자동 생성
- **AI Agent** — 캠페인 이상 탐지·자동 최적화 제안
- 예측 분석 및 실시간 알림

**리포팅**

- 화이트라벨 리포트 (클라이언트 브랜딩 적용)
- 실시간 KPI 대시보드

### 비즈니스 현황 (2025)

- Series B $1,300만 투자 유치
- 2027년 IPO 로드맵 공표
- 최근 출시: AI Agent, 확장된 통합 생태계, Creative Intelligence Suite

### 강점

- 광고 집행 데이터 통합 최강 (650+ 소스)
- 크리에이티브 단위 성과 분석 독보적
- 노코드로 낮은 진입장벽
- 화이트라벨로 에이전시 리셀러 모델 지원

### 약점

- 한국어 소셜/여론 데이터 없음
- 광고 성과 데이터에만 특화 (공론장 여론 분석 불가)
- Cloudflare 봇 차단 수준의 폐쇄적 접근 정책
- 비용 불투명 (엔터프라이즈 견적 방식)

## 5. AI SignalCraft의 차별화 포인트

### Sometrend가 답하지 못하는 것

> Sometrend: "이런 키워드가 이만큼 언급됐고, 감성은 이렇습니다."  
> AI SignalCraft: "이 여론 구조에서 위험 요인은 무엇이고, 어떤 프레임으로 대응해야 하며, 최악의 시나리오는 무엇인가."

- 수집 소스를 조직이 원하는 채널로 완전 교체 가능
- 분석 로직이 공개되어 팀 맞춤 프롬프트 엔지니어링 가능
- 민감 데이터가 외부 서버로 나가지 않음

### Adriel이 다루지 않는 영역

> Adriel: "이 광고 소재가 어떤 성과를 냈고, 어떻게 최적화할까."  
> AI SignalCraft: "공론장에서 이 이슈가 어떻게 확산되고, 어떤 프레임 전쟁이 벌어지고 있는가."

- 광고 성과가 아닌 **여론·공론장 데이터** 처리에 특화
- 한국 커뮤니티(DC갤러리, FM코리아 등) 수집 가능
- 온프레미스 운용으로 데이터 주권 확보
