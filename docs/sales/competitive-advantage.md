# AI SignalCraft — 경쟁 우위 & 커스터마이징 역량

> "표준 SaaS의 편리함, 전용 솔루션의 정밀함을 동시에"

---

## 1. 시장 내 포지셔닝

| 서비스             | 한 줄 정의                                               |
| ------------------ | -------------------------------------------------------- |
| **Sometrend**      | 한국 SNS 빅데이터 기반 소비자 트렌드 **조회** 서비스     |
| **Adriel**         | 멀티채널 광고 운영(AdOps) 통합 데이터 플랫폼             |
| **AI SignalCraft** | 공인 여론 자동 수집 + AI **다단계 전략 분석** 파이프라인 |

경쟁사가 데이터를 **보여주는** 데 집중한다면, AI SignalCraft는 데이터에서 **어떻게 대응할 것인가**까지 제시합니다.

## 2. 경쟁사 대비 차이점

### Sometrend(썸트렌드) 대비

| 구분             | Sometrend                  | AI SignalCraft                             |
| ---------------- | -------------------------- | ------------------------------------------ |
| **분석 방식**    | 키워드 빈도·감성 단순 집계 | 14개 모듈 다단계 AI 해석                   |
| **결과물**       | "이런 말이 많이 보였다"    | "이렇게 대응하라" (위험도·전략·시뮬레이션) |
| **데이터 소스**  | 플랫폼이 정한 채널 고정    | 수집 소스 완전 교체·추가 자유              |
| **AI 활용**      | AI 리포트 요약 (보조 기능) | Stage 1~4 분석 파이프라인 핵심 엔진        |
| **커스터마이징** | 플랜 내 제한적             | 모듈·프롬프트·리포트 형식 완전 조정        |
| **LLM 선택**     | 단일 고정 모델             | Claude·GPT·Gemini·DeepSeek 멀티 지원       |
| **배포**         | 퍼블릭 SaaS                | 온프레미스 운용 (데이터 외부 유출 없음)    |

> **핵심 차이**: Sometrend는 시장 조사·트렌드 파악 도구입니다.  
> AI SignalCraft는 여론에 **전략적으로 대응**하기 위한 분석 엔진입니다.

## 3. AI SignalCraft만의 핵심 강점

### ① 14단계 AI 분석 파이프라인

단순 감정 분류에서 끝나지 않습니다. Stage 1부터 4까지 순차적으로 깊어지는 분석 구조로, 수집된 여론이 **실행 가능한 전략 리포트**로 변환됩니다.

```
Stage 1 (병렬): 거시 동향 · 세그먼트 분석 · 감성 프레임 · 메시지 영향력
Stage 2 (순차): 리스크 지도 → 기회 발굴 → 전략 수립 → 종합 요약
Stage 4 (고급): 지지율 예측 · 프레임 전쟁 → 위기 시나리오 · 승리 시뮬레이션
```

### ② 멀티 LLM 지원 — 비용·성능 최적화

Claude · GPT · Gemini · DeepSeek 중 모듈별로 최적 모델을 선택합니다.  
예: 고비용 요약 모듈은 DeepSeek으로 교체 시 캐시 히트 57% 절감 가능.

### ③ 데이터 보안 — 조직 인프라 내 완전 격리

수집된 여론 데이터가 외부 SaaS 서버로 나가지 않습니다.  
민감한 정치·기업 이슈를 다루는 조직에 필수적인 구조입니다.

### ④ 오픈 파이프라인 — 블랙박스 없음

분석 로직이 완전히 투명하게 공개되어 있어 팀 맞춤 프롬프트 엔지니어링이 가능합니다.  
경쟁사가 제공하는 AI는 모두 블랙박스입니다.

## 4. 커스터마이징 역량 — 조직에 딱 맞는 분석 환경

AI SignalCraft는 표준 SaaS이면서 동시에 커스터마이징이 가능한 플랫폼입니다.  
소스, 모듈, 프롬프트, 리포트 형식 등 필요한 부분을 고객 요청에 맞게 조정합니다.

---

### 수집 소스 추가

기본 6개 소스 외에 특정 사이트, 내부 보고서, 해외 미디어 등 조직에서 모니터링하는 채널을 수집 소스로 추가할 수 있습니다.

- 특정 언론사·블로그 타겟 수집
- 해외 미디어 모니터링
- 내부 설문·인터뷰 데이터 연동

> 수집기는 `Collector` 인터페이스를 구현하면 파이프라인에 즉시 통합됩니다.  
> Sometrend가 정해진 채널만 제공하는 것과 달리, AI SignalCraft는 **분석 범위를 무한히 확장**할 수 있습니다.

---

### 분석 모듈·프롬프트 조정

기존 14개(고급 포함 시 59개) 모듈의 분석 관점, 출력 형식, 사용 AI 모델을 조직의 목적에 맞게 세밀하게 조정하거나 전용 모듈을 신규 제작합니다.

- 도메인 특화 분석 관점 적용 (의료·법률·금융·정치 등)
- 조직 내부 용어·분류 기준 반영
- 전용 AI 모듈 신규 개발

> 다른 솔루션은 AI 분석 결과를 받아볼 수만 있습니다.  
> AI SignalCraft는 **AI가 무엇을 어떻게 분석할지** 직접 정의합니다.

---

### 리포트 형식 맞춤화

출력 리포트의 섹션 구성, 시각화 방식, 브랜딩, 배포 형태를 고객사의 보고 체계에 맞게 커스터마이징합니다.

- 조직 브랜딩 적용 PDF 자동 생성
- 경영진용 요약본 별도 생성
- 슬랙·이메일 자동 발송 연동

> 리포트가 조직 내 보고 체계에 자연스럽게 녹아들어,  
> 분석 결과를 설득력 있게 전달하는 시간을 줄여줍니다.
