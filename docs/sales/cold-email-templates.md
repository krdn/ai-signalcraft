## 사용 가이드

### 핵심 원칙

1. **반드시 1페이지 여론 분석 샘플을 첨부**할 것 — 제네릭 콜드메일과의 최대 차별점
2. 사전에 AI SignalCraft로 해당 기업/인물 분석을 실행하여 실제 결과물 사용
3. 첫 메일은 화~목 오전 10~11시 발송이 최적

### 발송 일정

| 일차   | 액션                                 |
| ------ | ------------------------------------ |
| Day 0  | 첫 메일 (템플릿 1/2/3 중 선택)       |
| Day 3  | 팔로업 1 (미응답 시)                 |
| Day 7  | 팔로업 2 (최종)                      |
| Day 14 | 리스트에서 제거 또는 3개월 후 재시도 |

### 추적 지표

- 오픈율 목표: 40%+
- 응답률 목표: 10%+
- 데모 전환율 목표: 5%+
