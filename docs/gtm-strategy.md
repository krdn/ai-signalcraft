# AI SignalCraft — GTM(Go-To-Market) 전략

> 작성일: 2026-04-04 | 버전: 1.0

## 제품 정의

한국 온라인 여론(네이버 뉴스/댓글, 유튜브, DC갤러리, FM코리아)을 자동 수집하고 14개 AI 분석 모듈로 **실행 가능한 전략 리포트**를 생성하는 B2B SaaS 플랫폼.

**핵심 가치**: 수일 걸리던 여론 분석을 1~3시간으로 단축. 단순 감정 분석이 아닌 "어떻게 대응할 것인가"까지 제시.

---

## 1. ICP (이상적 고객 프로필)

### Tier 1 (최우선 공략)

**① 정치 캠프 / 선거 전략팀**

- 직책: 캠프 전략실장, 공보 담당관
- 팀 3~8명 / 예산 3,000만~2억원
- 페인 포인트: 여론조사 3~7일 소요, 실시간 온라인 반응 파악 불가
- 의사결정: 운영위/전략실장, **1~2주 내 빠른 결정**
- 킬러 모듈: `approvalRating`, `winSimulation`, `frameWar`
- **왜 최우선**: 긴급성 높고, 가격 민감도 낮고, 효과 입증 시 구전 빠름

**② PR/위기관리 에이전시**

- 직책: 어카운트 디렉터, PR 전략 팀장
- 팀 5~30명 / 도구 예산 500만~3,000만원
- 페인 포인트: 수동 클리핑에 주 10~20시간, "그래서 뭘 해야 하는데?" 답변 못함
- 의사결정: 팀장 검토 → 대표 승인, 2~4주
- 킬러 모듈: `crisisScenario`, `riskMap`, `strategy`
- **왜 최우선**: 리셀러 파트너로 확장 가능

### Tier 2

**③ 기업 홍보/평판 관리팀** — 예산 1,000만~5,000만원 / 결재 4~8주
**④ 정치/여론 컨설턴트** — 예산 100만~500만원 / 본인 즉시 결정

## 2. 가격 전략

| 티어             | 월가        | 포함 내용                                 | 타겟      |
| ---------------- | ----------- | ----------------------------------------- | --------- |
| **Starter**      | 49만원      | 1개 대상, 3소스, 8개 모듈, 월 4회, 3명    | 컨설턴트  |
| **Professional** | **129만원** | 3개 대상, 5소스, 14개 모듈, 월 12회, 10명 | **주력**  |
| **Campaign**     | 249만원     | 5개 대상, 무제한, API, 전담 CSM           | 정치 캠프 |

- 14일 무료 트라이얼 (카드 불요)
- 연간 결제 시 2개월 무료 (17% 할인)
- 파운더 딜: 첫 5개사 50% 할인 (6개월)
- **영업 미팅 시 고객 관련 인물 1회 무료 분석** 제공

**가격 정당화**: 모니터링 주니어 인건비 월 250~350만 vs Professional 129만

## 4. 세일즈 프로세스

```
리드 생성 → BANT 퀄리파이 → 맞춤 데모(사전 분석) → 14일 트라이얼 → 클로징
```

### 핵심 피치

> "기존 도구는 수집 + 감정 분석에서 멈춥니다. AI SignalCraft는 거기서 시작합니다.
> 프레임 전쟁 분석, 위기 시나리오, 승리 시뮬레이션 — 모니터링이 아니라 전략 도구입니다."

### 반대 의견 대응

| 반대        | 대응                                                |
| ----------- | --------------------------------------------------- |
| "비싸다"    | 주니어 인건비 250~350만 vs 129만 + 리포트 품질 비교 |
| "보안"      | 공개 여론만 수집, 한국 내 서버, 팀별 격리           |
| "AI 정확도" | AI가 재료 정리, 전문가가 최종 판단                  |
| "기존 도구" | 기존 = "무슨 말이 있다" / 우리 = "어떻게 할 것인가" |
